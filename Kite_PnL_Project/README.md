 📊 Kite PnL Project Analysis

## 📌 Description
This project analyzes stock trading data to calculate Profit & Loss (PnL) along with all applicable charges.

---

## 🚀 Features
- Data cleaning using Pandas
- Turnover calculation
- Brokerage calculation (capped at ₹20)
- STT, GST, SEBI, Stamp Duty calculation
- Gross and Net PnL calculation
- Instrument-wise summary

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- OpenPyXL

---

## 📂 Output
- Cleaned dataset
- Charges summary
- Final PnL report exported to Excel

---

## 📈 Key Learnings
- Financial data analysis
- Real-world charge calculations
- Data aggregation & grouping
- Excel automation using Python
