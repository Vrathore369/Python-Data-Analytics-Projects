🚚 Logistics Cost Analysis (ShopX Project)

## 📌 Description
This project validates courier charges by comparing actual charges with expected costs based on weight and delivery zones.

---

## 🚀 Features
- Data merging from multiple datasets
- Weight slab analysis (fixed + additional charges)
- Zone-based delivery cost calculation
- Detection of incorrect courier charges
- Final audit report generation

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Data Transformation
- Excel Reporting

---

## 📂 Output
- Expected vs Actual cost comparison
- Mismatch identification
- Final audit report in Excel

---

## 📈 Key Learnings
- Data merging techniques
- Business logic implementation
- Cost validation analysis
- Real-world logistics problem solving
