💉 Vaccination Portal Analysis

## 📌 Description
This project analyzes vaccination data and provides user-wise vaccination status and insights.

---

## 🚀 Features
- User ID-based search
- Vaccination status detection:
  - Fully Vaccinated
  - First Dose Pending
  - Second Dose Due / Overdue
- Date-based eligibility calculation
- Data visualization using charts

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Datetime

---

## 📂 Output
- User vaccination status
- Summary dataset
- Bar chart visualization

---

## 📈 Key Learnings
- Conditional logic implementation
- Date handling in Pandas
- Data visualization
- Real-world health data analysis
